# Product Growth & Retention Analytics

## Business Objective
Identify where users drop off in the customer journey, determine which user segments are most affected, evaluate retention patterns, and assess whether an experiment improved conversion performance.

## Core Analyses
1. Funnel Analysis
2. Segmentation Analysis
3. Cohort Retention Analysis
4. A/B Test Evaluation
5. Revenue Impact Estimation

## Business Questions
- Where are users dropping off most heavily?
- Which segments show the weakest conversion behavior?
- Are users retained over time after first purchase or first activity?
- Did the tested change improve conversion significantly?
- What action should the business prioritize?

- SQL analysis files
- Python scripts
- Visual outputs
- Business summary
